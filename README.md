# ONG-GROUP-6-PROJET-1

## À propos

Un site web pour une ONG (Organisation Non Gouvernementale). Le projet inclut une page d'accueil, une galerie de composants réutilisables, et différentes sections comme un agenda, des projets et un formulaire de contact.

## Structure du projet

```
├── index.html           # Page d'accueil principale
├── components.html      # Galerie de tous les composants
├── css/                 # Feuilles de styles
│   ├── index.css
│   ├── components/      # Styles des composants (boutons, cartes, etc.)
│   ├── sections/        # Styles des sections (header, footer, etc.)
│   └── tools/          # Styles utilitaires (reset, layout, typo)
└── ressources/
    └── img/            # Images du projet
```

## Comment commencer

1. Ouvre `index.html` dans ton navigateur pour voir la page d'accueil
2. Consulte `components.html` pour voir tous les composants disponibles
3. Les styles sont organisés par dossier CSS pour une meilleure maintenance

## Fonctionnalités

- 📋 Agenda des événements
- 🎯 Galerie de projets
- 📧 Formulaire de contact et newsletter
- 🎨 Composants réutilisables (boutons, cartes, inputs)
- 📱 Design responsive

## Technologies

- HTML5
- CSS3
- Structure modulaire et réutilisable
